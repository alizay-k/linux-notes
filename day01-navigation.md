# Day 1 — Terminal & Navigation

#Commands I learned today

pwd — prints where you are. Example: /home/user
ls — lists files. ls -l shows details. ls -a shows hidden files.
cd — changes directory. cd .. goes up. cd - goes back. cd ~ goes home.
clear — clears the screen.
history — shows past commands. !! repeats the last one.

# What the file system looks like

/ — root, everything starts here
/home — where user files live
/etc — config files
/var — logs and data
/tmp — temporary files
